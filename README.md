# gwf

A Google web font downloader

## Usage

Run with [Deno@^1.9.2](https://deno.land/)

```sh
deno run \
  --unstable \
  --allow-net \
  --allow-read \
  --allow-write \
  mod.ts d "https://fonts.googleapis.com/css2?family=Noto+Sans+SC:wght@400;700&family=Noto+Sans+TC:wght@400;700&display=swap"
```
